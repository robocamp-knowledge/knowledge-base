# RoboCamp Knowledge Base

This repository is the public, structured knowledge hub for RoboCamp.  
It serves as the central, authoritative source of information for:

- course descriptions and curriculum structure,
- licensing plans and product metadata,
- lesson-to-standard mapping (NGSS, ISTE, CCSS, CSTA, PL),
- structured blog content (posts and chapter-level JSON),
- archived social media content,
- video metadata and transcripts,
- JSON schemas for all knowledge objects.

It supports:
- AI assistants and agents (ChatGPT, Zapier, n8n, custom GPTs),
- SEO and content publishing,
- data consistency across all RoboCamp automation systems.

---

## Repository Structure

```
.github/               # workflows and CI configuration
blog/                  # articles and structured chapters (EN/PL)
  └── articles/        # individual blog articles
    ├── 3d-printing-in-school/
    ├── lego-science-review/
    └── lego-spike-prime-review/
offer/                 # product and educational offer data
  ├── catalog/         # pricing catalogs (PL/EN)
  ├── courses/         # course metadata and content
social/                # social media archive and posts
tools/                 # helper scripts, schemas and utilities
```
